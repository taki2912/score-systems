# 自动备份方案

## 🔄 方案1：定时自动备份（推荐）

### 使用方法：

**手动备份一次：**
```bash
cd "F:\Program Files\score-system"
python auto_backup.py
```

**自动备份（每10分钟）：**
双击运行 `start_auto_backup.bat`，保持窗口开着

### 特点：
- ✅ 每10分钟自动备份一次
- ✅ 保留最近20个备份
- ✅ JSON 格式，易读易恢复
- ✅ 自动清理旧备份

---

## 🎯 方案2：数据库触发器（实时备份）

需要在 PostgreSQL 中创建触发器，每次数据变动时自动通知。

**优点**：实时备份
**缺点**：配置复杂，需要额外服务监听

---

## 💡 推荐使用方案1

**原因：**
- 简单可靠
- 10分钟间隔足够频繁
- 不影响数据库性能
- 易于维护

---

## 📦 备份文件位置

```
F:\Program Files\score-system\backups\
  ├── backup_20260420_150000.json
  ├── backup_20260420_151000.json
  └── ...（保留最近20个）
```

---

## 🔧 Windows 计划任务（开机自动运行）

1. 打开"任务计划程序"
2. 创建基本任务
3. 触发器：系统启动时
4. 操作：启动程序
5. 程序：`F:\Program Files\score-system\start_auto_backup.bat`

这样电脑开机后会自动开始备份！
